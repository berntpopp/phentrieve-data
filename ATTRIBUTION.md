# Attribution

This repository distributes derived vector indexes for the Human Phenotype
Ontology (HPO). Each release specification records the exact upstream HPO
release tag, source URL, and SHA-256 digest used for its build.

- Ontology: [Human Phenotype Ontology](https://hpo.jax.org/)
- Source releases: [obophenotype/human-phenotype-ontology](https://github.com/obophenotype/human-phenotype-ontology/releases)
- Upstream license and attribution: [HPO repository license](https://github.com/obophenotype/human-phenotype-ontology/blob/master/LICENSE.txt)

Embedding models remain subject to their respective Hugging Face repository
licenses. Their exact revision hashes are recorded in every release
specification and bundle manifest.
